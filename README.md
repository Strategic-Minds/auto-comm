# AUTO COMM

Enterprise WhatsApp-first lead generation and governed AI assistant scaffold for the Xtreme Polishing Systems ecosystem.

## Canonical Stack

- Drive: canonical planning and builder documentation control plane.
- GitHub: implementation mirror and deployment source for Vercel.
- Vercel: dashboard, webhook endpoints, cron validation, and agent surfaces.
- n8n: automation workflow brain.
- Supabase: operational database, lead ledger, consent ledger, campaign state, and audit history.
- WhatsApp/Twilio: primary messaging channel using verified number +15559730487.
- AI Gateway / OpenAI: classification, lead scoring, response drafting, summaries, and validation agents.
- Base44 super agent: apex orchestrator for autonomous supervised build and swarm coordination.

## Governance Rule

This system may automate intake, classification, routing, reporting, and approved response generation. It must not bypass opt-in, opt-out, platform template rules, TCPA requirements, or human escalation rules for sensitive cases.

## First Build Objective

Bootstrap a controlled launch against prior Polished Concrete University attendees, then scale to contractor and location-based campaigns only after consent, deliverability, reply quality, and revenue metrics are proven.
