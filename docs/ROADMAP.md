# Roadmap

## Phase 1: Governed Foundation

Drive folder system, GitHub scaffold, Vercel dashboard shell, WhatsApp webhook route, Supabase schema mapping, and compliance docs.

## Phase 2: Initial Warm Launch

Import 500 prior PCU attendees, classify consent, launch 50-contact pilot, tune copy and response logic, then expand to full eligible list.

## Phase 3: Contractor Expansion

Build nationwide contractor list by territory, add location-specific promotions, segment by contractor type, and launch region-by-region campaigns.

## Phase 4: Digital Bidding

National Epoxy Pros bid intake, project qualification assistant, site/photo intake, estimator handoff, and quote status automation.

## Phase 5: Autonomous Scale

Vercel validation agents, AI Gateway routing, multi-location analytics, automated weekly campaign recommendations, and partner/referral network.
